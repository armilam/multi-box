+ (BOOL) addRegisteredUser: (MBUser*)  user

Adds a registered user to the collection. If the user is already represented in the list (identified by MBUser::userId), then the user is not added. This is normally called after authenticating a user.

Parameters
userThe user to add

A BOOL indicating whether the user was added to the collection

+ (void) authenticateNewUser: (NSDictionary*)  params
completion: (MBUser* newUser, NSException* error)  completion 

Gets a user's accessToken and refreshToken. Also populates the MBUser object with details about the user, including name, email address, and avatar image URL.

Parameters
paramsThe params returned from the Box API after a user logs in.
completionA block to be called once the user is authenticated (or authentication failed). The block passes in the new MBUser object and an NSException object which will be nil unless any errors occurred.

Provided by category MBUser(Authentication).

- (void) refreshAccessToken

Refreshes MBUser::accessToken. The Box API invalidates MBUser::accessToken every hour, so this must be called in order to have continued access. As long as MBUser::refreshToken is still valid, calling this will give the MBUser object a new valid MBUser::accessToken.

Provided by category MBUser(Authentication).

+ (NSArray *) registeredUsers

Returns an NSArray of MBUser objects representing all users currently registered in the app.

Returns
An NSArray of MBUser objects

Provided by category MBUser(Collection).

Revokes the MBUser object's MBUser::accessToken and MBUser::refreshToken. Useful on logout.
